# 数据库字典（精简后最终版）

## 1. 总体介绍

- 编写目的：说明精简后数据库结构，供开发、实施、运维统一参考。
- 适用范围：当前 `CCGL_MIG` 精简后库（业务对象 + 最小治理留痕对象）。
- 目录参考：`docs/仓储系统数据字典原始版本.doc`。

## 2. 概述
