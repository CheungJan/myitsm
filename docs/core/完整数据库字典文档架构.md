### 📊 建议的字典文档结构

#### **A. 概览部分**

- 数据库基本信息（版本、字符集、时区）
- 表总数、索引总数、约束总数
- 存储空间统计
- 业务模块分类统计

#### **B. 表详情部分**

按业务模块分组：

- **主数据模块**（tmm*）：客户、物料、设备等
- **ITSM模块**（tit*）：维护、翻新、开通等
- **仓储模块**（twh*）：入库、出库、库存等
- **采购模块**（tpc*）：计划、订单、结算等
- **销售模块**（tsl*）：订单、发货、结算等
- **财务模块**（tfn*）：账务、支付、资产等
- **系统模块**（tmc*）：用户、权限、参数等

#### **C. 关系图谱**

- ER图（主要表关系）
- 外键依赖关系
- 数据流向图

#### **D. 附录**

- 字段类型对照表
- 业务术语表
