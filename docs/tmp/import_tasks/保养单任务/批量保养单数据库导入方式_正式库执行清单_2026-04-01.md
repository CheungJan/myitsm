# 批量保养单数据库导入方式（正式库执行清单）

> 版本：v2026-04-01  
> 适用：正式库一次上线执行  
> 原则：先小批量验证，再导入剩余，失败同批次补跑。

---

## 0. 执行前准备

- 在系统命令行执行 `sqlplus`（不要在 `SQL>` 提示符下再输入 `sqlplus`）。
- SQL 语句统一使用英文符号（`,` `;`）。
- 本次批次号约定（示例）：
  - 小批量：`BY20260401_P_T01`
  - 剩余导入：`BY20260401_P_R01`

---

## 4. 小批量验证（10条）

```sql
/* 4.1 选10条未分批数据 */
UPDATE MIG_BY_IMPORT_STG T
   SET T.BATCH_ID = 'BY20260401_P_T01',
       T.STATUS = 'NEW',
       T.ERR_MSG = NULL,
       T.UPDATE_TIME = SYSDATE
 WHERE T.STG_ID IN (
       SELECT STG_ID
         FROM (
               SELECT STG_ID
                 FROM MIG_BY_IMPORT_STG
                WHERE BATCH_ID IS NULL
                ORDER BY STG_ID
              ) X
        WHERE ROWNUM <= 10
 );
COMMIT;

/* 4.2 执行导入 */
BEGIN
    PRC_IMPORT_BY_FROM_STG(
        P_BATCH_ID       => 'BY20260401_P_T01',
        P_CREATOR        => '2393',
        P_FORCE_START_NO => NULL
    );
END;
/

/* 4.3 验收 */
SELECT STATUS, COUNT(*) CNT
  FROM MIG_BY_IMPORT_STG
 WHERE BATCH_ID = 'BY20260401_P_T01'
 GROUP BY STATUS;

SELECT STG_ID, CUSTCARD, ENGINEER_ID, PLAN_DATE, GEN_DAILY_ID, ERR_MSG
  FROM MIG_BY_IMPORT_STG
 WHERE BATCH_ID = 'BY20260401_P_T01'
 ORDER BY STG_ID;
```

## 5. 小批量时间规则验收

```sql
/* 同工程师同日间隔异常（55~65分钟） */
WITH T AS (
    SELECT M.DAILY_MAINTENANCE_ID,
           M.STORE_ID,
           M.REQUEST_ENGINNER_ID,
           M.REQUEST_TIME,
           TRUNC(M.REQUEST_TIME) AS PLAN_DAY
      FROM TIT17_MAINTENANCE M
      JOIN MIG_BY_IMPORT_STG S
        ON S.GEN_DAILY_ID = M.DAILY_MAINTENANCE_ID
     WHERE S.BATCH_ID = 'BY20260401_P_T01'
       AND S.STATUS = 'SUCCESS'
),
X AS (
    SELECT T.*,
           ROUND(
             (REQUEST_TIME - LAG(REQUEST_TIME) OVER (
                PARTITION BY REQUEST_ENGINNER_ID, PLAN_DAY
                ORDER BY REQUEST_TIME
             )) * 24 * 60,
             2
           ) AS DIFF_MIN
      FROM T
)
SELECT *
  FROM X
 WHERE DIFF_MIN IS NOT NULL
   AND (DIFF_MIN < 55 OR DIFF_MIN > 65)
 ORDER BY REQUEST_ENGINNER_ID, PLAN_DAY, REQUEST_TIME;

/* 回访时间必须早于关单时间 */
SELECT STG_ID, GEN_DAILY_ID, GEN_RV_TIME, GEN_CLOSE_TIME
  FROM MIG_BY_IMPORT_STG
 WHERE BATCH_ID = 'BY20260401_P_T01'
   AND STATUS = 'SUCCESS'
   AND GEN_RV_TIME >= GEN_CLOSE_TIME;
```

> 以上两条均返回空集即可进入下一步。

## 6. 剩余数据导入

```sql
/* 6.1 打剩余批次 */

/* 6.2 执行剩余导入 */

        P_FORCE_START_NO => NULL

/

/* 6.3 结果检查 */
SELECT STATUS, COUNT(*) CNT
```

## 7. 失败补跑（同批次）

```sql
/* 7.1 数据修复后，将FAIL重置为NEW（可选） */
UPDATE MIG_BY_IMPORT_STG
   SET STATUS = 'NEW',
       ERR_MSG = NULL,
       UPDATE_TIME = SYSDATE
 WHERE BATCH_ID = 'BY20260401_P_R01'
   AND STATUS = 'FAIL';
COMMIT;

/* 7.2 重跑同一批次 */
BEGIN
    PRC_IMPORT_BY_FROM_STG(
        P_BATCH_ID       => 'BY20260401_P_R01',
        P_CREATOR        => '2393',
        P_FORCE_START_NO => NULL
    );
END;
/

/* 7.3 复核 */
SELECT STATUS, COUNT(*) CNT
  FROM MIG_BY_IMPORT_STG
 WHERE BATCH_ID = 'BY20260401_P_R01'
 GROUP BY STATUS;
```

## 8. 常见问题快查

## 8.1 `ORA-20014: 磁卡号未唯一匹配custcd`

```sql
SELECT CUSTCARD,
       COUNT(*) CNT_ALL,
       SUM(CASE WHEN USEFLG='1' THEN 1 ELSE 0 END) CNT_USE1
  FROM TMM22_CUSTOMERS
 WHERE CUSTCARD IN (
       SELECT CUSTCARD
         FROM MIG_BY_IMPORT_STG
        WHERE BATCH_ID = 'BY20260401_P_R01'
          AND STATUS = 'FAIL'
 )
 GROUP BY CUSTCARD;
```

- 建议治理主数据，保证过程口径下唯一有效。

## 8.2 磁卡号疑似有不可见字符

```sql
SELECT STG_ID,
       '[' || CUSTCARD || ']' AS CARD_SHOW,
       LENGTH(CUSTCARD) AS LEN_CHR,
       LENGTHB(CUSTCARD) AS LEN_BYTE,
       DUMP(CUSTCARD, 1016) AS HEX_DUMP
  FROM MIG_BY_IMPORT_STG
 WHERE BATCH_ID = 'BY20260401_P_R01'
   AND STATUS = 'FAIL';
```

## 9. 回滚（仅在必要时）

> 按批次回滚成功数据，顺序：子表 -> 主表 -> 暂存状态。

```sql
DELETE FROM TIT27_CLOSE_BILLS
 WHERE MAINTENANCE_ID IN (
   SELECT GEN_DAILY_ID FROM MIG_BY_IMPORT_STG
    WHERE BATCH_ID = 'BY20260401_P_R01' AND STATUS='SUCCESS'
 );

DELETE FROM TIT24_MAINTENANCE_RV
 WHERE MAINTENANCE_ID IN (
   SELECT GEN_DAILY_ID FROM MIG_BY_IMPORT_STG
    WHERE BATCH_ID = 'BY20260401_P_R01' AND STATUS='SUCCESS'
 );

DELETE FROM TIT23_MAINTENANCE_D2D
 WHERE MAINTENANCE_ID IN (
   SELECT GEN_DAILY_ID FROM MIG_BY_IMPORT_STG
    WHERE BATCH_ID = 'BY20260401_P_R01' AND STATUS='SUCCESS'
 );

DELETE FROM TIT17_CUST_POS_DAILY
 WHERE DAILY_MAINTENANCE_ID IN (
   SELECT GEN_DAILY_ID FROM MIG_BY_IMPORT_STG
    WHERE BATCH_ID = 'BY20260401_P_R01' AND STATUS='SUCCESS'
 );

DELETE FROM TIT17_MAINTENANCE
 WHERE DAILY_MAINTENANCE_ID IN (
   SELECT GEN_DAILY_ID FROM MIG_BY_IMPORT_STG
    WHERE BATCH_ID = 'BY20260401_P_R01' AND STATUS='SUCCESS'
 );

UPDATE MIG_BY_IMPORT_STG
   SET STATUS = 'ROLLED_BACK',
       UPDATE_TIME = SYSDATE
 WHERE BATCH_ID = 'BY20260401_P_R01'
   AND STATUS = 'SUCCESS';

COMMIT;
```

## 10. 交付记录建议

每轮执行后保留以下信息：

- 批次号（测试批次、剩余批次）
- `STATUS` 汇总截图
- `FAIL` 明细与修复记录
- 时间规则异常查询结果（是否空集）
- 最终是否达到 `SUCCESS` 全量通过
